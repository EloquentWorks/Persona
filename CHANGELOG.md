# 📝 Changelog

All notable changes to Laravel Persona will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project follows semantic versioning.

## [Unreleased]

### Added

- Documentation refresh with icon-led headings.
- Expanded release checklist and upgrade guidance.

### Changed

- Improved README structure to match the Eloquent Works package style.

## [1.0.0] - 2026-07-19

### Added

- Public profile model support.
- `HasPersona` user trait.
- Slug-based public profile URLs.
- Unique username support.
- Username change tokens.
- Configurable token intervals and maximum token balances.
- Display names, headlines, mottos, bios, and locations.
- Avatar and banner URL helpers.
- Website, social links, and custom links.
- Public/private visibility controls.
- Optional publishing support.
- Profile view tracking.
- Profile query scopes.
- Optional profile comments.
- Nested replies with configurable max depth.
- Publishable configuration.
- Publishable migrations.
- Publishable views.
- Optional built-in profile routes.
- PHPUnit, PHPStan/Larastan, Laravel Pint, and GitHub Actions setup.
- Documentation directory.
